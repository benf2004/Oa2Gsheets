OA2Gsheets requires you to link a Keepa account
Open OA2Gsheets -> Settings
Then, follow these instructions:
  1. Go to keepa.com/#!api
  2. Copy your "Private API access key"
    Note: your key is never shared with OA2Gsheets. You can always invalidate your key and create a new one by clicking the icon.
  3. Paste your key into the "Keepa API Key" box. Your key is now saved

Already linked and still getting this error? Make sure your Keepa key was copied correctly.
